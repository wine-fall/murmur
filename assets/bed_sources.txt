# murmur background-bed manifest (spec 03-04)

# One yt-dlp-resolvable ref per line; blank lines and lines starting with "#"
# are ignored. These are pulled to ~/.cache/murmur/bed/ at first-run loading
# (or via `make bed-refresh` after editing this file) and played as a low-volume
# instrumental backdrop under all talk. The repo stays binary-free: only this
# text list is committed; the audio lives in the per-user cache.

# Curation (L1): a small rotating set of calm, instrumental, no-vocal, ~1-hour
# radio-style mixes — long enough that the loop is inaudible, one per style
# (piano, ambient, lofi, classical). Refs are pinned watch URLs (reproducible),
# audio-only is pulled (yt-dlp -f bestaudio). Note on license: these are
# "relaxing / no copyright" mix channels; yt-dlp does not report a CC license
# for them (unlike a strict CC-BY set), but this is the same acquisition posture
# as the featured songs, which also stream from arbitrary YouTube (master §10.1).
# Swap in verified CC-BY / public-domain sources if a stricter posture is wanted.
